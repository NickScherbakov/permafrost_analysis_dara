Да, данные CALM по **Bykovsky** можно использовать, но **не как основной показатель для Якутска**, а только как сравнительный региональный фон.

Главная логика такая:

| Данные | Как использовать в ВКР по Якутску |
|---|---|
| **R42 Tuymada** | основной CALM-ряд для Якутского региона |
| **R43 Neleger** | основной CALM-ряд для Якутского региона |
| **R29A/R29B/R29C Bykovsky** | только сравнение с северной дельтой Лены |
| **ROS по Киренску, Олёкминску, Якутску, Жиганску** | можно использовать как профиль по Лене для гидрометеорологических рисков |

Почему Bykovsky не стоит включать в основной расчёт по Якутску:

- Bykovsky расположен в дельте Лены, около **71.8° с. ш.**;
- Якутск расположен около **62° с. ш.**;
- это разные мерзлотные, климатические и ландшафтные условия;
- глубина сезонного протаивания у Bykovsky около **24–42 см**, а у площадок Якутского региона — около **120–208 см**;
- если усреднить Bykovsky с R42/R43, показатель ALT для Якутска будет искажён.

Для основной части ВКР лучше считать ALT только по площадкам Якутского региона:

- **R42 Tuymada**;
- **R43 Neleger**.

Bykovsky можно вынести в отдельный подраздел как пример северных мерзлотных условий Якутии. Формулировка для текста ВКР может быть такой:

> Для оценки глубины сезонного протаивания в районе Якутска использованы данные CALM по площадкам R42 Tuymada и R43 Neleger, расположенным в Якутском регионе. Площадки Bykovsky R29A–R29C, находящиеся в дельте Лены, не использовались для расчёта локального показателя ALT для Якутска из-за существенных различий в широтном положении, мерзлотно-ландшафтных условиях и глубине сезонного протаивания. Эти данные рассматривались только как сравнительный материал для характеристики северных мерзлотных условий Якутии.

По ROS-событиям ситуация другая. Их как раз можно анализировать по профилю Лены:

- Киренск — 135 событий;
- Олёкминск — 70 событий;
- Якутск — 29 событий;
- Жиганск — 44 события.

Это методически оправдано, потому что для заторных и паводковых рисков важны не только условия в самом Якутске, но и процессы выше по течению Лены. Поэтому для ROS можно оставить цепочку **Киренск → Олёкминск → Якутск → Жиганск**, а для ALT использовать только локальные площадки Якутского региона.
